# Project Architecture

PromptLab is a Python and is an internal tool for AI engineers to **store, organize, and manage their prompts**. And complete backend code is in `/backend`:

- Entry point is `/backend/main.py`
- API routes in `/backend/app/api.py`
- Pydantic Models in `/backend/app/models.py`
- Data storage in `/backend/app/storage.py`
- tests in `/backend/tests`
